#### Installing Java on Linux

```bash
sudo apt-get update
sudo apt-get install -y python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install -y oracle-java8-installer
sudo update-alternatives --config java
```
